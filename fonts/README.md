# 字体目录使用说明

## 简介

本项目使用统一的字体管理系统，所有字体文件应放置在此目录中。

## 支持的字体格式

- `.ttf` - TrueType 字体
- `.ttc` - TrueType Collection 字体
- `.otf` - OpenType 字体
- `.woff` - Web Open Font Format
- `.woff2` - Web Open Font Format 2.0

## 如何添加字体

1. 将字体文件复制到此目录（`fonts/`）
2. 重启应用程序，系统会自动扫描并加载新字体
3. 字体将自动出现在"花字配置"和"水印配置"的字体下拉选项中

## 推荐的中文字体

- **微软雅黑** (`msyh.ttc`) - Windows 系统默认中文字体
- **黑体** (`simhei.ttf`) - 粗体中文字体
- **宋体** (`simsun.ttc`) - 传统中文字体
- **楷体** (`kaiti.ttf`) - 书法风格字体
- **仿宋** (`fangsong.ttf`) - 仿古风格字体

## Windows 系统字体位置

Windows 系统字体通常位于：
```
C:\Windows\Fonts\
```

您可以从这里复制需要的字体文件到 `fonts/` 目录。

## 注意事项

1. 字体文件名不要包含特殊字符或空格
2. 确保字体文件的版权允许您使用
3. 如果字体文件很大，可能会影响启动速度
4. 建议只添加项目实际需要的字体文件

## 故障排除

### 字体没有出现在下拉选项中

1. 确认字体文件已复制到 `fonts/` 目录
2. 确认字体文件扩展名正确（.ttf, .ttc, .otf 等）
3. 重启应用程序
4. 检查日志文件，查看是否有字体加载错误

### 字体显示异常

1. 确认字体文件完整且未损坏
2. 尝试使用其他字体文件
3. 检查字体是否支持您使用的字符（如中文、日文等）

## 示例

添加微软雅黑字体的步骤：

1. 打开 `C:\Windows\Fonts\` 目录
2. 找到 `msyh.ttc` 文件
3. 复制到项目根目录的 `fonts/` 文件夹
4. 重启应用程序
5. 在"花字配置"或"水印配置"中选择"msyh.ttc"字体

## 技术支持

如果遇到字体相关的问题，请查看日志文件：
```
logs/
```

日志中会显示字体加载的详细信息，包括成功加载的字体列表和任何错误信息。
